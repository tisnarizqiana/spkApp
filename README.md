**InfluencerMatch - Smart Decision Support System**

Aplikasi Fullstack yang berfungsi menganalisis dan merekomendasikan influencer terbaik untuk kampanye pemasaran. Menggunakan metode Profile Matching untuk menyeimbangkan aspek Kualitas (Core Factor) dan Popularitas (Secondary Factor).

Fitur Unggulan:
- Algoritma Cerdas: Perhitungan otomatis Gap Analysis, Bobot, dan Ranking.
- Dynamic Dataset: Upload data influencer via CSV secara real-time.
- Visualisasi Data: Grafik interaktif (Radar Chart & Bar Chart) untuk analisis mendalam.
- Comparison Tool: Fitur Head-to-Head untuk membandingkan kandidat secara langsung.
- Professional Reporting: Export hasil rekomendasi ke dalam laporan PDF resmi.
- Secure System: Autentikasi lengkap (Login, Register, Reset Password via Email OTP) dengan keamanan JWT & Bcrypt.
